# Glovity 使用者操作手册

版本日期：2026-07-04

本文档面向实际使用、演示、测试和基础维护人员。仓库首页即为使用者手册；如需离线查阅，可下载根目录中的 [Word 版使用者手册](./Glovity%20使用者操作手册_v20260704.4.docx)。

本仓库是面向用户的正式发布包，只保留正式使用需要的内容。Arduino 工程源码、旧版独立校准流程、本地链路测试工具、Reference 资料和内部资料不包含在本仓库中。

## 2. 使用前需要准备什么

正式使用通常需要以下设备和软件：

| 项目 | 用途 |
|---|---|
| 左手手套 | 采集左手动作数据 |
| 右手手套 | 采集右手动作数据 |
| 无线接收器 | 接收左右手数据并连接电脑 |
| 软件主程序 | 显示手部动作和参数界面 |
| 数据线或充电线 | 给设备供电、充电或进行维护操作 |

电脑环境要求：

| 项目 | 要求 |
|---|---|
| 操作系统 | 以软件部门发布的版本要求为准，当前主要按 Linux 使用 |
| 显卡 | 需要 NVIDIA GPU |
| 驱动环境 | 需要 CUDA 驱动环境 |
| 软件目录 | `01_Software/glove-sdk` |

## 3. 正常启动流程

1. 确认左右手手套和无线接收器属于同一套产品。
2. 给左右手手套上电。
3. 将无线接收器连接到电脑。
4. 等待电脑识别出接收器端口。
5. 进入软件主程序目录：

```bash
cd 01_Software/glove-sdk
```

6. 运行软件主程序：

```bash
./glove-sdk
```

7. 在软件中选择或确认接收器端口。
8. 观察左右手是否开始实时动作显示。

端口编号会随电脑、插口、系统重启而变化，这是正常现象：

| 系统 | 端口示例 |
|---|---|
| Windows | `COM3`、`COM12`、`COM27` |
| Linux | `/dev/ttyACM0`、`/dev/ttyACM1` |

不要把某一次测试时看到的端口号当成固定值。实际使用时，以软件识别到的设备列表或当前系统显示为准。

## 4. 首次姿态校准

每只手套第一次使用前，建议各做一次姿态校准。更换电脑、重新部署软件、删除校准文件或姿态明显不准时，也需要重新校准。

在软件目录中打开终端：

```bash
cd 01_Software/glove-sdk
```

左手校准示例：

```bash
./glove-sdk --calibrate --port /dev/ttyACM0 --side left --text
```

右手校准示例：

```bash
./glove-sdk --calibrate --port /dev/ttyACM0 --side right --text
```

其中 `/dev/ttyACM0` 只是 Linux 示例端口，实际使用时请替换为当前连接的手套端口。Windows 环境下端口通常显示为 `COMx`，例如 `COM12`。

校准时按软件提示完成三步动作：

1. 手掌朝下水平，手指自然伸直并略微张开，按 Enter。
2. 手腕向上转约 90 度，指尖朝上，按 Enter。
3. 手腕向下转约 90 度，指尖朝下，按 Enter。

校准完成后，软件目录中会生成校准文件：

```text
imu_calibration_left.npy
imu_calibration_right.npy
```

这些文件会影响姿态显示效果，请不要随意删除、移动或覆盖。

## 5. 正常使用时的软件界面

启动后，软件通常包含两类区域：

| 区域 | 作用 |
|---|---|
| 动作显示区 | 实时显示手部动作，可用鼠标旋转、缩放或平移视角 |
| 参数控制区 | 调整方向、坐标预设、手指联动、张开角度、手腕修正等参数 |

常见控制项：

| 控件 | 用途 |
|---|---|
| `Flip Direction` | 翻转手指弯曲方向或相关坐标方向 |
| `Coordinate Preset` | 切换坐标预设 |
| `DIP Coupling` | 调整末端指节跟随中间指节的比例 |
| `Finger Splay` | 微调手指张开角度 |
| `Wrist Yaw Fix` | 修正手腕水平旋转偏差 |
| `Reset Parameters` | 将界面参数恢复为默认值 |
| `FPS` | 显示当前刷新率 |

如果动作显示方向不对，优先使用界面中的方向和坐标相关控件调整。不要通过重新烧录设备程序来处理普通显示方向问题。

## 6. 磁力计维护校准

磁力计校准不是开机必须步骤，也不是首次拿到产品时必须执行的步骤。研发人员已经在出厂前完成初始磁力计校准。

建议在以下情况进行维护校准：

- 每半个月到一个月做一次例行维护校准。
- 手套在固定姿态下方向持续漂移。
- 使用过程中出现明显数据偏移或方向不稳定。
- 更换使用环境后，方向明显不稳定。
- 技术人员要求做一次完整维护校准。

当前发布包已经支持通过电脑工具触发磁力计校准，不需要用户手动切换设备程序。

进入工具目录：

```powershell
cd 03_Maintenance_Tools\Integrated_Mag_Calibration
```

只检查当前连接的设备：

```powershell
python .\integrated_mag_calibrate.py --auto --scan-only
```

校准左手：

```powershell
python .\integrated_mag_calibrate.py --auto --side left --duration 35
```

校准右手：

```powershell
python .\integrated_mag_calibrate.py --auto --side right --duration 35
```

左右手依次校准：

```powershell
python .\integrated_mag_calibrate.py --auto --all --duration 35
```

操作时请按以下步骤进行：

1. 连接需要校准的手套。
2. 运行对应命令。
3. 按提示开始校准。
4. 在约 35 秒内缓慢旋转手套，让它覆盖多个方向。
5. 等待工具显示完成和保存结果。
6. 校准完成后，手套会回到正常工作状态。

校准过程中，正常动作数据可能短暂停止，这是正常现象。请不要在保存完成前断开连接或关闭设备。

## 7. 设备烧录和配对

生产或维护人员需要给一套设备烧录和配对时，使用：

```text
02_Device_Flasher/Glovity_Flasher.exe
```

该工具是单文件烧录器，内嵌左手、右手和无线接收器三份正式下载固件。使用时将三块设备连接到 Windows 电脑，在工具界面中选择左手、右手、接收器对应端口，工具会自动读取设备标识、完成配对并烧录。

空白设备无法自动判断左右手，需要测试人员根据物理标记手动选择。烧录过程中不要拔出设备，也不要打开串口助手、Arduino IDE 或其他可能占用端口的软件。

## 8. 电量和自动关机

手套会自动监测电量状态。

当电池电压持续低于 3.50V 时，手套会自动关机，用于保护电池和设备。

如果手套电量正在下降，并且连续 15 分钟没有收到无线接收器响应，也会自动关机，避免长时间空耗电。

如果手套正在连接电脑或充电设备，电量通常不会继续下降，这种情况下不会因为没有接收器响应而自动关机。但只要电压已经低于 3.50V，仍会触发低电量保护。

建议使用时关注以下现象：

| 现象 | 处理方式 |
|---|---|
| 手套突然关机 | 先充电，再重新上电 |
| 长时间未连接接收器后关机 | 属于省电保护，重新上电即可 |
| 低电量时动作显示不稳定 | 先充电，不建议继续采集或演示 |

## 9. 常见现象和处理办法

| 现象 | 说明或处理 |
|---|---|
| 端口号和上次不一样 | 正常。端口号由电脑系统分配，以当前识别结果为准 |
| 接收器端口直接打开后不是可读文本 | 正常。接收器输出给软件读取，不适合用普通串口窗口人工查看 |
| 软件找不到校准文件 | 先做首次姿态校准，或确认 `.npy` 文件仍在软件目录中 |
| 只有一只手有动作 | 检查另一只手是否上电、是否属于同一套产品、是否完成校准 |
| 左右手显示反了 | 优先检查软件中的左右手设置和当前连接设备 |
| 方向慢慢漂移 | 先确认电量充足；仍明显漂移时做磁力计维护校准 |
| Linux 下提示端口权限不足 | 将当前用户加入串口权限组后重新登录，或联系维护人员处理 |
| 软件无法启动或画面异常 | 确认电脑具备 NVIDIA GPU 和 CUDA 驱动环境 |

## 10. 使用注意事项

- 使用同一套产品中的左右手手套和无线接收器，不要混用其他套装中的设备。
- 不要删除软件目录中的 `.npy` 校准文件。
- 不要在磁力计校准保存完成前断开连接。
- 不要用普通串口窗口判断接收器数据是否正常。
- 不要在低电量状态下继续长时间演示或采集。
- 未经技术人员确认，不要更换设备程序或修改软件目录中的关键文件。

## 11. 推荐使用顺序

第一次拿到一套产品时，建议按以下顺序操作：

1. 确认左右手手套和无线接收器为同一套产品。
2. 给左右手手套充电。
3. 连接无线接收器并启动软件。
4. 分别完成左右手首次姿态校准。
5. 正常启动动作显示。

磁力计校准不属于首次开机必做步骤。出厂前已经完成初始校准；后续只需要按半个月到一个月的维护周期执行，或在出现明显方向漂移、数据偏移时执行。

日常使用时，通常只需要：

1. 给左右手手套上电。
2. 插入无线接收器。
3. 启动软件主程序。
4. 选择或确认接收器端口。
5. 开始使用。
