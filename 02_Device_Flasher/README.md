# Glovity 设备烧录器

本目录用于生产烧录和配对一套 Glovity 设备。

## 文件

```text
Glovity_Flasher.exe
```

该文件是单文件烧录器，内嵌三份正式下载固件：

| 设备 | 固件用途 |
|---|---|
| 左手手套 | 采集左手动作并发送到配对接收器 |
| 右手手套 | 采集右手动作并发送到配对接收器 |
| 无线接收器 | 接收配对左右手数据并输出给软件 |

## 使用步骤

1. 在 Windows 电脑上双击 `Glovity_Flasher.exe`。
2. 如果浏览器没有自动打开，手动访问 `http://127.0.0.1:18080`。
3. 将左手手套、右手手套和无线接收器同时连接到电脑。
4. 在页面中为三个端口分别选择左手、右手和接收器。
5. 点击读取并烧录按钮。
6. 等待工具完成设备标识读取、配对信息写入和烧录。
7. 完成后，这三块设备就是一套绑定产品。

## 注意事项

- 空白设备无法自动判断左右手，需要测试人员根据物理标记手动选择。
- 烧录前关闭串口助手、Arduino IDE 或其他可能占用端口的软件。
- 烧录过程中不要拔出设备。
- 本用户发布包不包含 Arduino 工程源码；正式烧录请使用本目录中的单文件烧录器。
